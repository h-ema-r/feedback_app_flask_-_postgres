# Python Feedback App

> Python Flask Feedback app that sends data to Postgres database and emails user

## Quick Start

```bash
# Add your DATABASE URI in app.py and your mail params in send_mail.py

# Install dependencies
pipenv shell
pipenv install

# Serve on localhost:5000
python app.py
```
